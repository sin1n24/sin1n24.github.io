# Engineering Ideas into Reality.

[**🌐 公開ウェブサイトを表示する**](https://sin1.studio/)

Hydejack（Jekyllテーマ）を使用して構築された、個人のポートフォリオサイトのリポジトリです。製作したロボット、ソフトウェア、VRプロジェクト、および主催している企画をまとめています。

## 🚀 主な製作物

| プロジェクト名 | 概要 | 技術スタック |
| :--- | :--- | :--- |
| **Links** | リンク機構シミュレータ・設計補助ソフト | `C++`, `DxLib`, `Jules`, `Inverse Kinematics` |
| **Leopard** | かわさきロボット競技大会出場ロボット | `Robotics`, `ESP32`, `EasyEDA`, `Mechanism` |
| **VR遠隔操縦システム** | thetaとVRゴーグルによるロボット搭乗体験 | `Unity`, `Robotics`, `Mechanism` |
| **かわロボVR** | 物理演算を伴うVR空間内ロボット操縦練習ソフト | `Unity`, `Android`, `Mechanism` |
| **ORBITAL_DRIFT** | 高速軌道アクションゲーム | `Vanilla JS`, `Canvas 2D` |

## 📅 企画・イベント

- **かわロボアドベントカレンダー**: 技術交流を目的とした恒例の執筆イベント。
- **ミニかわロボ**: 手のひらサイズの格闘対戦ロボット競技規格の策定と大会運営。

## 🛠 技術スタック

- **SSG**: [Jekyll](https://jekyllrb.com/)
- **Theme**: [Hydejack](https://hydejack.com/) (Free Version)
- **Customization**:
  - `my-inline.scss`: ボタンのスタイリングや、画像へのホバーアニメーション、要素のフェードイン処理を実装。
  - `my-head.html`: カスタムスクリプトやメタデータの追加。

## 🎨 特徴

- **アニメーション**: CSS（SCSS）によるリッチなホバーエフェクトとフェードイン実装。
- **レスポンシブ**: モバイル・デスクトップ両対応。
- **ダークモード**: OS設定に連動した配色切り替え。
