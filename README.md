# Engineering Ideas into Reality.

[**🌐 公開ウェブサイトを表示**](https://sin1.studio/)

Hydejack（Jekyllテーマ）を使用した個人のポートフォリオサイトのリポジトリ。製作したロボット、ソフトウェア、VRプロジェクト、および主催企画のまとめ。

## ロボット
+ **Leopard**: かわさきロボット競技大会出場機体。独自のリンク機構設計と制御基板。
+ **VR遠隔操縦システム**: 自作ロボットに搭乗する代わりにTHETAとVRゴーグルで搭乗感覚を再現。

## ハードウエア
+ **小型ジョイスティックコントローラ**: M5ATOMおよびStickに対応した、最大3ボタン2軸の小型ジョイスティックコントローラ。スイッチサイエンスにて委託販売中。
+ **M5Stack向 サーボ接続基板**: M5StickC/Atom/Capsule各シリーズに対応したサーボ接続基板。スイッチサイエンスにて委託販売中。

## ソフトウエア
+ **Links**: リンク機構シミュレータ兼かわロボ設計補助ソフト。AIを活用したウェブアプリへの移植。
+ **かわロボVR**: 物理演算のあるVR空間でのロボット操縦練習、試運転、およびネット対戦。Quest/Android対応。
+ **ORBITAL_DRIFT**: 高専時代に制作した初のアクションミニゲームのAIによる再現。

## 企画
+ **かわロボアドベントカレンダー**: クリスマス前に有志でかわロボの記事を執筆する恒例イベント（毎年開催）。技術交流会も実施。
+ **ミニかわロボ**: 手のひらサイズの格闘対戦ロボット競技規格。ミニ大会を随時開催。
